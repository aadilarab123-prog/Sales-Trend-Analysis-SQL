## 📌 Project Overview

This project focuses on analyzing sales trends using SQL aggregation techniques. The objective is to extract meaningful business insights from sales data by performing revenue analysis, order volume analysis, time-based reporting, and advanced SQL analytics using window functions.

The project is built using *MySQL 8.0* and demonstrates SQL skills commonly required for Data Analyst roles.

---

## 🎯 Objectives

- Analyze monthly and yearly sales performance.
- Calculate revenue and order volume.
- Identify top-performing products and periods.
- Perform trend analysis using window functions.
- Generate business reports using SQL.

---

## 🗂️ Dataset

### Table: online_sales

| Column | Data Type | Description |
|---------|-----------|-------------|
| order_id | INT | Unique Order ID |
| order_date | DATE | Date of Order |
| product_id | INT | Product Identifier |
| amount | INT | Order Amount |

Dataset Size:
- *100 Sales Records*
- Multiple Years of Sales Data
- Multiple Products
- Monthly Sales Information
